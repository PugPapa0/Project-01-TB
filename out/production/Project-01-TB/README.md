# Daily Sugar Intake Tracker

## Project Overview
The Daily Sugar Intake Tracker is a Java console application that allows users to input five foods consumed during a day along with their sugar content in grams. The program calculates total sugar intake and compares it against a recommended daily limit of 36 grams based on the American Heart Association guideline.

---

## How the Program Works
1. The user enters their name.
2. The user enters five food items.
3. The user inputs sugar content for each food.
4. The program calculates total sugar intake.
5. The program compares the total against a daily recommended limit.
6. The program displays a formatted report and health evaluation.
